SOLUCIONES OBLIGATORIAS PARA UN DATA LAKE REAL
------------------------------------------------
1. S3 Lifecycle Policy
   - Bronze  → Glacier después de 90 días  ($0.004/GB)
   - Silver  → Standard-IA después de 30 días ($0.0125/GB)
   - Gold    → Standard permanente ($0.023/GB)

2. Particionado por fecha
   - Permite eliminar particiones antiguas sin borrar todo

3. Compactación de archivos
   - Consolida muchos parquet pequeños en archivos grandes
   - Reduce PUT requests y mejora rendimiento de lectura

4. TTL (Time To Live) por capa
   - Bronze:  90 días → luego Glacier o borrado
   - Silver:  1 año   → luego archivar
   - Gold:    indefinido (solo agregaciones)

5. Usar Glacier Deep Archive para históricos
   - $0.00099/GB (vs $0.023 Standard = 96% menos)
   - Ideal para datos que casi nunca se consultan
